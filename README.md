# prolog-99-problems
"My solutions of “Ninety-Nine Prolog Problems” crafted by Werner Hett at the Berne University of Applied Sciences in Berne, Switzerland. Implemented using SWI-Prolog."

Este repositorio contiene mis soluciones implementadas en **SWI-Prolog** para la famosa colección de **99 Problemas de Programación Lógica** (Prolog Problems), originalmente creada por Werner Hett.

# Origen de los Problemas

Los enunciados de los problemas son tomados y adaptados de la colección canónica disponible en:

* **[ttybitnik/lab.99-problems](https://github.com/ttybitnik/lab.99-problems)**
